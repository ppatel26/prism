# Styles (the skin)

Each style is a folder with a `style.yaml` (source of truth, validated against `../schema/style.schema.json`) and an optional `README.md` with example renders.

## Families

- **A - Journal / Whitepaper.** Serious, dense, text + figure. The depth champions.
- **C - Editorial Data-Journalism.** Premium, chart-led, polished.
