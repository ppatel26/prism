# Zoom (the depth)

Zoom is a numeric level-of-detail, NOT a fixed set of tiers. Wire it to a density target (panel count / element count / word budget) so a dense topic can be pushed as far as it needs.

Suggested scale (0-5):

| Level | Feel | Rough density |
|-------|------|---------------|
| 0 | napkin | one idea, a few shapes |
| 1 | sketch | one simple diagram |
| 2 | working | a real labeled diagram |
| 3 | brief | a few panels |
| 4 | poster | many panels (most posters here) |
| 5 | full poster | maximum legible density |

## The legibility ceiling

Past level 5 you do NOT keep cramming. You SPILL into a multi-image series (poster 1 of N), which is just Shneiderman's "details on demand" realized as a sequence. The render pipeline's image-count decision handles this.

**Reference:** Shneiderman, "Overview first, zoom and filter, then details on demand."
